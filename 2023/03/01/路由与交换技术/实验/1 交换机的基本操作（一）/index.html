交换机的基本操作

2023/3/1 路由与交换技术 路由与交换技术实验

实验目的：

掌握交换机的常用操作命令及配置方法，包括交换机Console口令、TELNET口令、用户名和密码的配置。

实验拓扑结构：

实验过程：

1. 配置计算机与交换机的ip地址
1.1 将计算机与交换机连接

首先用配置线将计算机与交换机连接，如图1.1所示；

（1） 计算机选择RS232；
（2） 交换机选择Console.


1.1 拓扑结构
1.2 设置计算机和交换机ip地址
1.2.1 计算机配置ip地址；

依次点击：PC0—Desktop—IP Configuration，配置ip地址，如图1.2所示（本次实验不涉及网关和DNS）。


图1.2 计算机ip地址
1.2.2 配置交换机管理ip地址；

首先将计算机进入终端，依次点击：PC0—Desktop—Terminal—确定，如图1.3所示。


图1.3 计算机终端

然后依次输入如下命令：

Switch>enable
Switch#configure terminal
Switch(config)#interface vlan 1
Switch(config-if)#ip address 192.168.1.254 255.255.255.0
Switch(config-if)#no shutdown
Switch(config-if)#exit

如图1.4，ip地址配置成功。


图1.4 交换机管理ip地址配置
1.3 配置用户名和密码
1.3.1 配置console密码

输入如下命令，配置console密码。

Switch>enable
Switch#configure terminal
Switch(config)#line console 0
Switch(config-line)#password cisco
Switch(config-line)#login
Switch(config-line)#login local
Switch(config-line)#exit
Switch(config)#username Switch password switch
Switch(config)#username Switch privilege 15 password switch
Switch(config)#
Switch#

如图1.5所示：


图1.5 配置console
1.3.2 特权模式密码（全局密码）

输入如下命令，配置特权模式密码。

Switch>enable
Switch#configure terminal
Switch(config)#line console 0
Switch(config-line)#password cisco
Switch(config-line)#login
Switch(config-line)#login local
Switch(config-line)#exit
Switch(config)#username Switch password switch
Switch(config)#username Switch privilege 15 password switch
Switch(config)#
Switch#

结果如图1.6所示。


图1.6 配置特权模式密码
1.3.3 配置虚拟通道密码（TELNET）

输入如下命令，配置特权模式密码。

Switch#configure terminal
Switch(config)#line vty 0 5
Switch(config-line)#password telnet
Switch(config-line)#login
Switch(config-line)#login local
Switch(config-line)#exit
Switch(config)#exit
Switch#

结果如图1.7所示。


图1.7 配置虚拟通道密码
