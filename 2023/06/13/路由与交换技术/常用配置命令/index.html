exit

4. 单臂路由

int f0/24

exit

exit

int f0/0
no shutdown

ip address 192.168.10.1 255.255.255.0
exit

exit

5.RIP动态路由协议配置

router rip 
network 192.168.50.0
network 192.168.60.0
network 192.168.70.0
version 2   (设置RIPv2)
Router rip  
no auto-summary  （关闭RIP动态路由协议自动聚合功能）
exit
router rip 
redistribute static （路由重分发）

6. 单区域OSPF动态路由协议配置

（创建路由器环回接口）
int loopback 1
ip address 1.1.1.1 255.255.255.0
exit
router ospf 1
network 10.1.1.0 0.0.0.255 area 0
show ip ospf database   （查看OSPF数据库）
show ip ospf neighbor    （显示与OSPF相关的邻居信息）
show ip ospf   
debug ip ospf events    （检测OSPF事件）
debug ip ospf packet    （检测OSPF包的传输）

7. PPP协议及验证 多区域OSPF动态协议配置

username cisco password 123
int s0/0
encapsulation ppp
ppp authentication pap
exit
int s0/0
encapsulation  ppp
ppp pap sent-username cisco password 123
end
conf t
Username router password 123
int s0/1
encapsulation ppp
ppp authentication chap
(config-if)#
username  router password 123
int s0/1
encapsulation ppp
ppp authentication chap
启动路由器 3的远程管理功能（telnet服务），配置访问用户名为user，访问密码为password。
clock rate 64000
line vty 0 4
login local
int s1/1
ppp authentication chap
exit

8. ACL配置

router ospf 1
network 172.25.50.0 0.0.0.255 area 0
redistribute static subnets（重分发静态路由）   
redistribute connected subnets（重分发直连路由）
access-list 20 deny 10.40.20.0 0.0.0.255
access-list 20 permit any
int f1/0
ip access-group 20 out

10. VPN的配置

interface tunnel 0
ip address 192.168.10.2 255.255.255.0
tunnel source fastEthernet 0/0
                             源地址
tunnel destination 20.4.50.1
                             目标地址
exit
ip route 10.4.0.0 255.255.0.0 192.168.10.1
interface tunnel 0
ip address 192.168.10.1 255.255.255.0
tunnel source fastEthernet 0/0
tunnel destination 20.4.70.2
exit
ip route 172.16.0.0 255.255.0.0 192.168.10.2

11. 多区域进行连通性测试

enable
configure terminal
router ospf 100
area 1 virtual-link 192.168.70.1  （area 0 跨 area1 与 area 2 进行联通）
被跨区域                           对端地址
area 1 virtual-link 192.168.50.1  （area 2 跨 area1 与 area 1 进行联通）
