VLAN基础

2023/6/8 路由与交换技术 路由与交换技术

本节学习内容：

  1. 产生背景
  2. 构建vlan
  3. 划分vlan
  4. vlan优点

1. 产生背景

网络广播风暴问题。
一个企业内不同部门的信息传输问题。

2. 构建vlan

构建vlan可以使物理上不联系的主机，在逻辑上是连续的。

3. 划分vlan

  1. 基于端口的vlan划分
  2. 基于MAC地址的vlan划分
  3. 基于网络层的vlan划分

4. vlan优点

  1. 避免引发安全问题
  2. 成本降低
  3. 性能提高
  4. 分割广播域，防范网络广播风暴
