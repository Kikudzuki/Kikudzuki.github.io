静态路由

2023/5/30 路由与交换技术 路由与交换技术实验

实验目的：

  1. 掌握静态路由的基本配置方法；
  2. 掌握复杂拓扑结构静态路由配置方法；
  3. 掌握默认路由的配置方法。

图一：实验拓扑结构图

实验过程：

IP 192.168.10.100
掩码 255.255.255.0
网关 192.168.10.254 
IP 192.168.20.100
掩码 255.255.255.0
网关 192.168.20.254
IP 192.168.55.100
掩码 255.255.255.0
网关 192.168.55.254
IP 192.168.60.100
掩码 255.255.255.0
网关 192.168.60.254
Switch>enable
Switch#configure terminal
Switch(config)#vlan 10
Switch(config-vlan)#vlan 20
Switch(config-vlan)#exit
Switch(config)#interface f0/1
Switch(config-if)#switchport access vlan 10
Switch(config-if)#interface f0/11
Switch(config-if)#switchport access vlan 20
Switch(config-if)#interface f0/24
Switch(config-if)#switchport mode trunk
Switch(config-if)#end
Switch#
Router>enable
Router#configure terminal
Router(config)#interface f0/0.1
Router(config-subif)#encapsulation dot1q 10
Router(config-subif)#ip address 192.168.10.254 255.255.255.0
Router(config-subif)#exit
Router(config)#interface f0/0.2
Router(config-subif)#encapsulation dot1q 20
Router(config-subif)#ip address 192.168.20.254 255.255.255.0
Router(config-subif)#exit
Router(config)#interface f0/1
Router(config-if)#ip address 192.168.20.254 255.255.255.0
Router(config-if)#exit
Router(config)#ip route 192.168.32.0 255.255.224.0 192.168.30.1
Router(config)#end
Router#
Router>enable
Router#configure terminal
Router(config)#interface f0/0
Router(config-if)#ip address 192.168.30.1 255.255.255.0
Router(config-if)#interface f0/1
Router(config-if)#ip address 192.168.50.2 255.255.255.0
Router(config-if)#interface f0/1/0
Router(config-if)#ip address 192.168.40.2 255.255.255.0
Router(config-if)#exit
Router(config)#ip route 192.168.10.0 255.255.255.0 192.168.30.2
Router(config)#ip route 192.168.20.0 255.255.255.0 192.168.30.2
Router(config)#ip route 192.168.55.0 255.255.255.0 192.168.50.1
Router(config)#ip route 192.168.60.0 255.255.255.0 192.168.40.1
Router(config)#end
Router#
Router>enable
Router#configure terminal
Router(config)#interface g0/0
Router(config-if)#ip address 192.168.55.254 255.255.255.0
Router(config-if)#interface g0/1
Router(config-if)#ip address 192.168.50.1 255.255.255.0
Router(config-if)#exit
Router(config)#ip route 192.168.10.0 255.255.255.0 192.168.50.2
Router(config)#ip route 192.168.20.0 255.255.255.0 192.168.50.2
Router(config)#ip route 192.168.30.0 255.255.255.0 192.168.50.2
Router(config)#ip route 192.168.40.0 255.255.255.0 192.168.50.2
Router(config)#ip route 192.168.60.0 255.255.255.0 192.168.50.2
Router(config)#end
Router#
Router>enable
Router#configure terminal
Router(config)#interface f0/0
Router(config-if)#ip address 192.168.60.254 255.255.255.0
Router(config-if)#interface f0/1
Router(config-if)#ip address 192.168.40.1 255.255.255.0
Router(config-if)#exit
Router(config)#ip route 192.168.0.0 255.255.224.0 192.168.40.2
Router(config)#ip route 192.168.50.0 255.255.255.0 192.168.40.2
Router(config)#ip route 192.168.55.0 255.255.255.0 192.168.40.2
Router(config)#end
